## Requirements
1. Node 18
2. NPM 9.6.6
   

## How To Run
1. Clone the repo
   ```sh
   git clone https://github.com/Zukashi/hex-ocean.git
   ```
2. Change directory
   ```sh
   cd hex-ocean
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the script
   ```sh
   npm run dev
   ```
